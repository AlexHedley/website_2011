Syntax Highlighter

So I had seen some syntax highlighting done on other sites and wondered how it had been achieved.

I came across SyntaxHighlighter by Alex Gorbatchev.

Firstly you will either have to download the source code or link to it.

                <script src="scripts/shCore.js"></script>
                <link rel="stylesheet" href="css/shCore.css">
            
                <link rel="stylesheet" href="css/shCoreDefault.css">
            
                <script src="scripts/shBrushXml.js"></script>
                <script src="scripts/shBrushJScript.js"></script>
            
            <script>
                    SyntaxHighlighter.all();
            </script>

Now wrap any code in the <pre></pre> tags.

You then need to choose what language the code is so it will be styled accordingly.
Add class="brush: BRUSH ALIAS;" to the pre tag. 

<pre class="brush: js">
    /**
    * SyntaxHighlighter
    */
    function foo()
    {
        if (counter <= 10)
            return;
        // it works!
    }
</pre>

Finally you need to convert any html to the respective Entities.

Greater than: > is "&gt;"

Less than: < is "&lt;"

Check out the list of brushes and the aliases for the class name you need when using it on a pre tag.

                <script src="scripts/shBrushNAME.js"></script>

There is an autoloader function but I haven't been able to get this to work.

You can also use the <script>tag

<script type="syntaxhighlighter" class="brush: js"><![CDATA[
    /**
    * SyntaxHighlighter
    */
    function foo()
    {
        if (counter <= 10)
            return;
        // it works!
    }
]]></script>

Note: This takes the extra type="syntaxhighlighter" and the necessary <![CDATA[...]]>
